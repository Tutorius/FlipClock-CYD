There is a type of displays sold on AliExpress since 2025. These displays are complete Computers with an ESP32-chip and a touch-display 2,4".
The PCB-color is yellow, they are cheap, so this tells the name.

This software is for a display that was bought in may 2026. It touchpad and display share the same SPI-connection. Older tutorials from Random Ner5d do not apply for this display.

How the clock looks

The clock does not use any WLAN or Bluetooth. So it needs to be set by hand.
When starting, the numbers show four Zeros. By typing (with the pencil, or your fingertip) on the upper flip-area, the number of this digit is increased by one.
Typing or clicking on the lower flip-area descreses the number.
Time can be set between 00:00 and 23:59. Decreasing a zere leads to a 9, 5 or 2 (depending on the digit).
Start the clock by typing in thed left black area below the two first digits.
